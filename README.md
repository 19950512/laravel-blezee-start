# Hello World com Laravel Breeze + docker

## Required
- docker-compose instalado

## Usage

### Windows

Habilite execuções com Powershell
- Abrir o PowerShell como administrador
- Definir a política de execução
```powershell
  Set-ExecutionPolicy RemoteSigned
```
```Powershell
  ./start.ps1
```
